# Accessibility

Accessibility is not a theme skin. Contrast, focus, structure, and redundant
status cues determine whether a Pathfinder sheet is usable under stress.

## Concepts

- [focus keyboard print](/oks/pathfinder-character-sheet-design/accessibility/focus-keyboard-print.md) — operable without a mouse; printable without losing data.
- [color not only signal](/oks/pathfinder-character-sheet-design/accessibility/color-not-only-signal.md) — pair hue with text, icons, or patterns for status.
