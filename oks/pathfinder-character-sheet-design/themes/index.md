# Themes (Light and Dark)

Light and dark modes are not inverses of each other. Each needs its own surface
stack, elevation language, and verified contrast. Character sheets are
**long-session, low-media** UIs — exactly the class of product where dual themes
pay off if done carefully.

## Concepts

- [light mode surfaces](/oks/pathfinder-character-sheet-design/themes/light-mode-surfaces.md) — paper, panels, shadows, and positive polarity.
- [dark mode surfaces](/oks/pathfinder-character-sheet-design/themes/dark-mode-surfaces.md) — dark grays, elevation lightening, desaturated accents.
- [dual-theme strategy](/oks/pathfinder-character-sheet-design/themes/dual-theme-strategy.md) — when to invest and how to structure tokens.
- [system preference sync](/oks/pathfinder-character-sheet-design/themes/system-preference-sync.md) — default to OS settings; optional manual override.
