# Accent and Theme Identity

Character sheets evangelize the game’s mood, but Pathfinder play still needs
**numbers first**. Identity should live in surfaces, headers, rules (borders),
and a single strong accent — not in rainbow coding every subsystem.

## Patterns

- **One primary accent** for actions and key chrome (this project’s PF1-inspired
  maroon on parchment is a classic pairing).
- **Neutral surfaces** for long reading (skills lists, feat text).
- **Icons sparingly** for section recognition; Roll20 invested in system-aligned
  iconography only after information architecture stabilized.
- **Minimal decoration** on combat panels; flourish belongs in headers or empty
  biography space.

## Dual-theme caution

The parchment-and-maroon identity is a **light-theme story**. A dark theme should
reinterpret the same roles (paper → elevated surface, ink → primary text, accent
→ desaturated interactive red) rather than inverting hex values blindly. See
[dual-theme strategy](/oks/pathfinder-character-sheet-design/themes/dual-theme-strategy.md).
