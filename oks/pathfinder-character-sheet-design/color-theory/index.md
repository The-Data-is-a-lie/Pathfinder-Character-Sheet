# Color Theory

These concepts are **general UI color practice**. Character sheets inherit them
when choosing parchment/ink palettes, status colors (HP, buffs, errors), and
theme tokens. Prefer semantic roles over one-off hexes.

## Concepts

- [color roles and tokens](/oks/pathfinder-character-sheet-design/color-theory/color-roles-and-tokens.md) — map ink/paper/accent/danger to CSS variables.
- [contrast and WCAG](/oks/pathfinder-character-sheet-design/color-theory/contrast-and-wcag.md) — luminance contrast floors for text and controls.
- [saturation and harmony](/oks/pathfinder-character-sheet-design/color-theory/saturation-and-harmony.md) — how vivid color behaves on light vs. dark surfaces.
